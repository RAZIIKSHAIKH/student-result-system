
# Student Result Management System

A Python console application to manage student academic records.

## Features
- Add, update, delete and search student records
- Auto-calculates total marks, percentage and grade (A/B/C/D/F)
- Menu-driven interface with input validation

## How to Run
1. Install Python 3
2. Run: python result_system.py

## Technologies Used
- Python 3
- File Handling
- Exception Handling
